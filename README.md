# Instrucciones para ejecutar el aplicativo:

***Frontend***:
1. Instalación de librerias en /front-end/tutorial-canciones:
        ***npm install***
2. Ejecutar en /front-end/tutorial-canciones:
        ***ng serve***

***Backend***:
1. Ejecución docker compose:
        ***docker-compose-up***
